## Projects

### 🐉 [DragonEye-Auditor](https://github.com/Alok943/DragonEye-Auditor)
> Hinglish toxicity classifier built as a reinforcement learning environment

Built for the **Meta × Scaler OpenEnv Hackathon** — an RL environment that trains and evaluates AI agents on content moderation for Indian e-commerce reviews. Supports English, Hindi, and Hinglish with sarcasm/nuance detection.

- 3 graded tasks with difficulty progression: language ID → moderation → sarcasm detection
- Deterministic reward function bounded [0, 1] with task-conditioned episode sampling

- Deployed live on HuggingFace Spaces with Dockerfile and OpenAI-compatible inference script

### 🎬 [Cinematch V2](https://github.com/Alok943/cinematch_v2)
> Hybrid movie recommender that separates search from discovery

Most recommenders conflate title search with semantic discovery — Cinematch solves this with a two-stage retrieval pipeline. Stage 1 handles exact and typo-tolerant title matching. Stage 2 delivers semantic recommendations via sentence embeddings with genre-aware filtering and hybrid ranking.

- Two-stage retrieval: robust title matching + embedding-based semantic search
- Noise filtering via similarity thresholds, genre overlap constraints, duplicate removal
- Hybrid ranking combining semantic similarity, popularity, and recency
- Trailer integration via TMDB API with YouTube fallback

`ChromaDB` `Sentence Transformers` `Streamlit` `TMDB API` `Python`
